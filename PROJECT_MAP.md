# Project Map

## Configuration Helpers

- `py/griptape_settings.py` — отвечает за загрузку пользовательских настроек Griptape внутри ComfyUI; при инициализации гарантирует наличие каталога `user` рядом с `users.json`, чтобы API-ключи и прочие параметры всегда сохранялись даже на «чистых» установках.

## Documentation

- `README.md` — основной гид по установке, обновлению и устранению неполадок; рядом с секциями Troubleshooting описано автоматическое создание пользовательского профиля.
